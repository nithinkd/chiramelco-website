# Chiramel & Co Website Redesign Project

**Project Goal**  
Clone and significantly improve https://chiramelco.org — make it modern, fast, professional, beautiful, and fully responsive.

- Outdated design, poor mobile UX, basic layout

- Content is mostly static and text-heavy
